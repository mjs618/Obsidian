## 🐍 Python 开发相关

### 启动开发服务器
```powershell
# 进入 backend 目录并启动 uvicorn 开发服务器
# --reload: 代码变更时自动重启
# --host 0.0.0.0: 允许外部访问
# --port 8000: 指定端口

cd E:\project\care\backend; python -m uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

### 停止 Python 进程
```powershell
# 简单停止所有 Python 进程
Get-Process -Name python -ErrorAction SilentlyContinue | Stop-Process -Force

# 优雅停止 Python 进程，显示进程信息
# cd e:\project\care; Get-Process python -ErrorAction SilentlyContinue | ForEach-Object { Write-Host "Stopping PID $($_.Id)..."; $_ | Stop-Process -Force }; Write-Host "All stopped" -ForegroundColor Green
```

## 📊 日志查看命令

### 查看错误日志
```powershell
# 实时查看最后 100 行错误日志
Get-Content backend/logs/error.log -Tail 100

# 使用 tail 命令查看错误日志（需要 WSL 或 Git Bash）
tail -n 100 backend/logs/error.log
```

### 查看应用日志
```powershell
# 查看应用日志并过滤包含 "patients" 或 "ERROR" 的行
# -Context 2: 显示匹配行前后各 2 行上下文
Get-Content backend/logs/app.log -Tail 200 | Select-String -Pattern "patients|ERROR" -Context 2
```

## 🧰 其他实用命令

### 文件和目录操作
```powershell
# 递归查找包含特定文本的文件
Get-ChildItem -Recurse | Select-String -Pattern "关键词"

# 批量重命名文件
Get-ChildItem *.txt | Rename-Item -NewName {"new_" + $_.Name}
```

### 系统信息查看
```powershell
# 查看系统信息
systeminfo

# 查看磁盘空间
Get-PSDrive -PSProvider FileSystem

# 查看网络连接
netstat -an
```

### Git 相关
```bash
# 查看 Git 状态
git status

# 提交更改

git push
```
